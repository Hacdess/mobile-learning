## 1. Setup environments:

### 1.1. JDK

Install via Oracle page: https://www.oracle.com/java/technologies/downloads/

### 1.2. Android Studio:

- Install it here: https://developer.android.com/studio

**SDK Platform:**

- Install at least 1 SDK.

- Please include Android SDK Platform Sources for Android.

- Please verify your PC configuration to install the required SDK components in the 'Package Details' section.

**SDK Tools:**

- Install SDK build-tool

- Install Android Emulator

- Install Android SDK Platform-Tools

- You can use Genymotion instead of Android Emulator: https://www.genymotion.com/
